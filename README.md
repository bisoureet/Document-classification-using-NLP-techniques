# Document-classification-using-NLP-techniques
This project applies Natural Language Processing (NLP) techniques to automatically classify documents into predefined categories. The workflow includes text preprocessing (tokenization, stopword removal, stemming/lemmatization), feature extraction using methods such as TF-IDF or word embeddings, and training classification models like Naive Bayes, Logistic Regression, or Support Vector Machines. The system is evaluated using accuracy, precision, recall, and F1-score to ensure robust performance.
The project demonstrates how NLP can transform unstructured text into structured insights, making document classification scalable and efficient for real-world applications such as spam detection, sentiment analysis, and topic categorization.
